# 🌿 AgroSmart — Cultivando o Futuro Verde

Site educativo sobre **agricultura sustentável**, com quiz interativo, seções informativas e design responsivo.

---

## 📋 Sobre o Projeto

O AgroSmart é uma landing page educativa que apresenta práticas de agricultura sustentável de forma acessível e engajante. O objetivo é conscientizar sobre o impacto da agricultura no meio ambiente e apresentar alternativas responsáveis para produção de alimentos.

---

```
agrosmart/

└── README.md       # Este arquivo
```

## 🌐 Seções da Página

| Seção | Descrição |
|---|---|
| **Hero** | Apresentação com estatísticas globais sobre agricultura |
| **Sobre** | O que é agricultura sustentável (6 pilares em cards) |
| **Práticas** | 6 boas práticas explicadas em detalhe |
| **Quiz** | 10 perguntas interativas com feedback e pontuação |
| **Rodapé** | Links de navegação e contato |

---

## 🧠 Quiz

O quiz possui **10 perguntas** de múltipla escolha sobre temas como rotação de culturas, compostagem, irrigação por gotejamento, controle biológico e agrofloresta. Ao final, o usuário recebe uma pontuação e uma mensagem personalizada em três níveis:

- 🌱 **0–4 acertos** — Ainda há muito para aprender!
- 🌿 **5–7 acertos** — Bom trabalho!
- 🏆 **8–10 acertos** — Guardião da Terra!

## 🎨 Design

- **Paleta:** Verde profundo, terracota, bege e areia
- **Fontes:** Playfair Display (títulos) + DM Sans (corpo)
- **Responsivo:** Layout adaptado para mobile com menu hambúrguer
- **Animações:** Reveal ao rolar a página (IntersectionObserver), header compacto ao fazer scroll

## 🚀 Como Usar

Não requer instalação ou dependências. Basta abrir o `index.html` em qualquer navegador moderno.

```bash
# Clone ou baixe os arquivos e abra diretamente:
open index.html
```

> As fontes são carregadas via Google Fonts — requer conexão com a internet para renderização correta.
